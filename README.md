# Psycho Slack

## Dependencies

[nodejs](https://nodejs.org/en/)

```sh
npm install
```

## Test

### Set local config

create a config file (config/default.json)

Config File example
```json
{
  "botToken": "xoxb-19879915587-JHHptmmE4SbpIQys6OOrftFy",
  "botName": "marc-bot",
  "testUser": "marc"
}
```

### Start the server
```sh
node index.js
```
